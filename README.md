# as86-assembler

This is a VS Code extension of syntax highlight for as86 assembler.

## Features

- syntax highlight for as86 assembler

## Requirements

- vscode >= 1.51.0

## Release Notes

### 0.1.0

as86 assembler syntax highlight，inlcude：

- register
- numeric
- label
- strings
- instructions
- operator
